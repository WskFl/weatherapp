# Weatherapp
An simple app to get data from the weatherapi.com

# Setup
1. Download and exctract Zip in an empty folder

2. Download Python 3.10 or higher

3. Open terminal and type
   
```
pip install -r requirments.txt
```
4. Go to weatherapi.com and signup, go to weatherapi.com/my/ and copy your api key

5. Paste your api key between the two "" at the API_KEY field

6. Run the script
```
python Weatherapp.py
```
